# otis

Optimized Toolkit for Interfacing Sleeper

```
otis/
├── app.R            # Main app file
├── global.R         # Global variables and functions loaded before the app
├── R/               # Custom R functions
├── data/            # Data files
├── www/             # Static assets (CSS, JS, images)
├── tests/           # Unit tests
├── inst/            # Additional files to be installed with the package
├── man/             # Documentation
├── DESCRIPTION      # Package metadata
├── NAMESPACE        # Package namespace directives
└── README.md        # Repository documentation
```
